How to Auto Follow/Unfollow/Scrape

  1. Open the web browser and go the the account or the post that you want to follow/unfollow/scrape its Followers, Following or Likes

    • Example account: https://www.instagram.com/natgeo/
    • Example post: https://www.instagram.com/p/BrUpd4AgEe3/

  2. You can choose to follow/unfollow/scrape the Followers or Following of a specific account, or the Likes of a specific post:

    • For the Followers: Click on his followers number to open the Followers popup, in our example: 95.3m followers

    • For the Following: Click on his following number to open the Following popup, in our example: 132 following

    • For the Likes: Open the post in a new tab, then click on the likes number to open the Likes popup, in our example: 50,340 likes

  3. Open the extension (click on the icon at the top right corner)

  4. Choose the mode (FOLLOW, UNFOLLOW or SCRAPE), change the configuration if you want to, then click on START

  5. [FOR SCRAPE MODE ONLY] You can export the accounts in different formats: XLS, CSV, TXT and HTML. Simply click on the format you want in the Export bar:
IMPORTANT NOTE:

  • It is mandatory to use it in a new window without any other tab, and do not minimize that window. If you use it in a normal tab and loose the visibility, Twitter and Instagram will not load the accounts.
  • Do not touch anything on the the page while the software is running.
  • If you click on STOP while it is loading accounts or in a sleeping state, you should refresh the current page before starting it again.
  • If you cannot load more accounts, check the javascript console, if you see an error 429, it means you reached the Instagram / Twitter rate limit, so please wait 30 - 90 minutes.
  • You can also scrape the usernames of a post comments, but you have to click on "Load more comments" to load the comments first. This feature is in BETA, for Instagram only.

Whitelist: You can add usernames in the whitelist so you don't follow or unfollow them again (you can activate "Auto whitelist" to add them automatically), you have to separate each username with a comma, example: instagram,natgeo,natgeowild

How to use multiple sessions/accounts

The extension is made to work with the current connected Instagram session on the web browser.
But there is a “trick” to use it with more than 1 session, go to Extensions (on the web browser), click on Details on my extension, and Allow in incognito.
So you can use it on 2 sessions per profile, then you can add more profiles: https://support.google.com/chrome/answer/2364824?hl=en
